Reproductor Player Music

Obtener ayuda

Busca información de ayuda y procedimientos para tu versión del Reproductor de Player Music. Obtén información sobre las funciones y cómo solucionar problemas.

Si Reproductor de Player Music ha dejado de funcionar correctamente después de las últimas actualizaciones, comprueba que las actualizaciones son el problema mediante Restaurar sistema. Para hacerlo:

  1. Selecciona el botón Inicio y luego escribe restaurar sistema.
  2. Selecciona Crear un punto de restauración y, después, en el panel Propiedades del sistema, selecciona Restaurar sistema.
  3. Sigue las indicaciones para seleccionar el punto de restauración precisamente anterior a la ronda de actualizaciones que parece haber desencadenado el problema. Luego, ejecuta el proceso de restauración del sistema. (Esto puede tardar un cierto tiempo).
  4. Después de restaurar el sistema a la configuración anterior, comprueba si el Reproductor Player Music vuelve a ejecutarse correctamente.
